# apt-sources.list
Utility to manage APT sources.list files

# Sample of use

## List available sources.list
```
apt-sources.list -l
```

## Install (but not activate) all sources.list

```
apt-sources.list --installsample
```

## Enable wheezy backports

```
apt-sources.list wheezy-bpo on
```
